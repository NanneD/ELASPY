BASE_LOCATIONS_FILE
===================

CSV file that specifies at which nodes the ambulance bases are located. The table below contains an example. In the example, bases are located at nodes 4, 8, 14 and 18.

.. list-table:: Example BASE_LOCATIONS_FILE.
   :widths: 5
   :header-rows: 1

   * - Base Locations
   * - 4
   * - 18
   * - 8
   * - 14

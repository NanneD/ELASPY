HOSPITAL_FILE
=============

CSV file that specifies at which nodes the hospitals are located. The table below contains an example. In the example, hospitals are located at nodes 1, 2 and 10.

.. list-table:: Example HOSPITAL_FILE.
   :widths: 5
   :header-rows: 1

   * - Hospital
   * - 1
   * - 10
   * - 2
